# 📱 Install Master C++ with uPhumeh - Installation Guide

## How to Install the App on Your Device

Master C++ with uPhumeh is a **Progressive Web App (PWA)** that can be installed on any device and used like a native app - even offline!

---

## 🖥️ Desktop Installation (Windows, Mac, Linux)

### Google Chrome / Microsoft Edge / Brave
1. Open the app in your browser: **http://localhost:3000** (or your deployed URL)
2. Look for the **install icon** (⊕ or download icon) in the address bar (right side)
3. Click the install button
4. Click **"Install"** in the popup dialog
5. The app will open in its own window
6. Find the app in your Start Menu (Windows) or Applications folder (Mac)

### Alternative Method:
1. Click the **three dots menu** (⋮) in the top-right corner
2. Select **"Install Master C++ with uPhumeh"** or **"Add to Desktop"**
3. Confirm the installation

---

## 📱 Mobile Installation

### Android (Chrome / Samsung Internet / Edge)
1. Open the app: **http://localhost:3000** (or your deployed URL)
2. Tap the **three dots menu** (⋮) in the top-right corner
3. Tap **"Add to Home screen"** or **"Install app"**
4. Name the app (default: "Master C++ with uPhumeh")
5. Tap **"Add"** or **"Install"**
6. Find the app icon on your home screen
7. Launch it like any other app!

### iPhone/iPad (Safari)
1. Open the app in **Safari**: **http://localhost:3000** (or your deployed URL)
2. Tap the **Share button** (square with arrow pointing up) at the bottom
3. Scroll down and tap **"Add to Home Screen"**
4. Name the app (default: "C++ EduHub")
5. Tap **"Add"** in the top-right corner
6. The app icon will appear on your home screen

**Note:** On iOS, you MUST use Safari browser for installation to work!

## 🗑️ Uninstall the App

### Desktop:
- **Windows**: Right-click the app icon → Uninstall
- **Mac**: Drag the app to Trash from Applications folder
- **Chrome**: Go to `chrome://apps` → right-click → Remove

### Mobile:
- **Android**: Long-press the app icon → Uninstall/Remove
- **iPhone**: Long-press the app icon → Remove App

## 🌐 For Deployment (Production)

When you deploy the app to a real domain (e.g., cppedu.com):

1. **Requirements:**
   - HTTPS is required (PWA won't install on HTTP)
   - Valid SSL certificate
   - Service worker must be served over HTTPS

2. **Installation will work on:**
   - Your custom domain: `https://yourdomain.com`
   - Any subdomain: `https://app.yourdomain.com`

3. **Users can install from:**
   - Desktop browsers (Chrome, Edge, Firefox, Safari)
   - Mobile browsers (Chrome, Safari, Samsung Internet)
   - QR code scanning
   - Direct link sharing

## ❓ Troubleshooting

**Install button not showing?**
- Make sure you're using a supported browser (Chrome, Edge, Safari)
- Check if the app is already installed
- Try refreshing the page (Ctrl+R or Cmd+R)
- On mobile, make sure you're not in incognito/private mode

**Can't find the installed app?**
- **Windows**: Check Start Menu or Desktop
- **Mac**: Check Applications folder or Launchpad
- **Android**: Check app drawer or home screen
- **iPhone**: Check all home screen pages

**Need help?**
Contact uPhumeh or check the documentation in the app!
